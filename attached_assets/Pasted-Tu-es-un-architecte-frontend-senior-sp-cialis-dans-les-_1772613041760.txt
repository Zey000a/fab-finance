Tu es un **architecte frontend senior spécialisé dans les plateformes fintech, banques digitales et services de crédit**.

Ta mission est de concevoir **un site web complet pour une entreprise de financement / crédit**, similaire aux plateformes professionnelles de prêt utilisées par les institutions financières.

Le site doit reproduire **la structure complète d’un site de service de prêt réel** : pages, sections, contenus, parcours utilisateur, simulation de prêt et présentation des services.

Le site doit inspirer **confiance, sérieux et professionnalisme**, comme une véritable institution financière.

---

# CONTRAINTE ABSOLUE

Le projet doit être **100% FRONTEND**.

Il est **STRICTEMENT INTERDIT** d'utiliser :

• backend
• base de données
• API serveur
• endpoints
• appels fetch vers un serveur
• stockage distant

Toutes les données doivent être :

• statiques
• simulées
• stockées uniquement côté frontend

Utiliser uniquement :

• mock data
• state React
• localStorage

Le site doit fonctionner **entièrement côté navigateur**.

# OBJECTIF DU SITE

Créer un site professionnel permettant :

• présenter les différents types de prêts
• expliquer les services financiers
• simuler un prêt
• faire une demande de prêt
• contacter l’entreprise

Le site doit reproduire **l’expérience utilisateur d’un vrai site bancaire ou de crédit**.

# MULTILINGUE (OBLIGATOIRE)

Le site doit être **entièrement multilingue**.

Langues supportées :

Français (FR)
Anglais (EN)
Allemand (DE)
Espagnol (ES)
Italien (IT)
Portugais (PT)

Utiliser :

react-i18next
ou un système JSON équivalent.

Les traductions doivent être stockées dans :

/locales/fr.json
/locales/en.json
/locales/de.json
/locales/es.json
/locales/it.json
/locales/pt.json

Exemple :

{
"home_title": "Une approche différente du financement",
"loan_request": "Demander un prêt",
"contact": "Contact"
}

Tous les textes du site doivent être traduisibles.

Aucun texte ne doit être codé en dur.

# SIMULATEUR DE PRÊT

Créer un simulateur interactif avec :

slider montant
slider durée
taux d’intérêt

Calculer :

mensualité
total intérêts
total remboursé

Formule :

mensualité = capital × taux / (1 − (1 + taux)^−durée)

Les résultats doivent se mettre à jour **en temps réel**.
