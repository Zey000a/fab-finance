# PROBLÈME 1 — LE SIMULATEUR A ÉTÉ SUPPRIMÉ

Le simulateur de prêt est **une fonctionnalité obligatoire du site**.

Il doit être restauré.

Le simulateur doit permettre :

• choisir le montant du prêt
• choisir la durée
• définir le taux

Afficher en temps réel :

• mensualité
• total remboursé
• total des intérêts

Formule :

mensualité = capital × taux / (1 − (1 + taux)^−durée)

Le simulateur doit être présent :

• sur la page principale
• ou sur une page dédiée
